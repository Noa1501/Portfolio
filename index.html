Développeur Python sur Odoo (Stage)

D.One Marseille, France
Avril - Juillet 2024 (4 mois)

Durant ce stage de fin d'études, j'ai intégré l'équipe de développement Odoo chez D.One, où j'ai pu mettre en pratique mes compétences en développement Python. J'ai été chargé de concevoir et développer une extension Odoo innovante pour l'activation automatisée de cartes SIM via QR Code.

Missions & Réalisations

  • Développement complet d'une extension Odoo pour l'activation automatisée de cartes SIM via QR Code
  • Conception de l'architecture logicielle et des modèles de données pour le système d'activation
  • Implémentation d'un système sécurisé de génération et de validation de QR Codes
  • Intégration avec les API des opérateurs télécom pour l'activation des cartes SIM
  • Création d'interfaces utilisateur intuitives pour le suivi des activations
  • Documentation complète du code et des fonctionnalités pour faciliter la maintenance

Résultats du projet

Le système d'activation automatisée de cartes SIM développé a permis de :

  • Réduire de 70% le temps d'activation des cartes SIM
  • Diminuer considérablement les erreurs d'activation manuelle
  • Améliorer l'expérience utilisateur pour les clients finaux
  • Fournir des statistiques détaillées sur les processus d'activation
